What is TXV

A format built for text.

⬡

Plain text, always

A .txv file is valid UTF-8. Open it in Notepad, VS Code, or cat it in a terminal. Every frame is right there, readable, copy-pasteable, human-writable.

◈

Colour-native

Every character has a hex colour stored alongside it. Only non-black cells are stored — so dark frames are tiny. Full RGB, greyscale, or rainbow sweeps.

▤

Animated by design

Frames stack sequentially with a header-defined FPS. The format is the text-native GIF — made for loops, recordings, and live camera output.

⌘

Open forever

No binary blobs, no proprietary encoding. Released into the public domain. Anyone can implement a parser, build a tool, or extend the format.

The Format

Self-describing. Self-contained.

A .txv file starts with a header block, then one frame block per frame, each separated by ---. That's the whole structure.

  • Header declares size, fps, frame count, and colour mode
  • Each frame: exactly ROWS lines of COLS characters
  • Colours stored as col,row,#rrggbb — non-black cells only
  • Unknown header keys are silently ignored — forward compatible
  • If a line is shorter than COLS, remainder is treated as spaces
  • Full parser spec: TXV_SPEC.md ↗
#txv v1
#size 90x52
#fps 24
#frames 120
#mode color
#author ben
---
#frame 1
`.,--iil|/\oosa@W
`,,--iil|/\oosa@W
`.,--iil|/\##@@WM
#colors
0,0,#c43a1b
1,0,#c43a1b
7,0,#00ff41
2,1,#ff8800
---
#frame 2
.,`--iil|/\oosa@W
...
---

Reference Tools

Four tools. Zero dependencies.

All browser tools run entirely in your browser — nothing is uploaded, nothing leaves your machine.

ASCII CAM browser
ascii-cam.html

Live webcam feed converted to character art in real time. Colour, B&W, Rainbow, Edge, and Glitch modes. Built-in .txv recorder with frame limit control.

↓ download
TXV PLAYER browser
txv-player.html

Drag and drop any .txv file to play it back. Reverse, ping-pong, glitch effect, 5 colour theme overrides, frame scrubber, speed control, and frame export.

↓ download
TXV CONVERTER browser
txv-converter.html

Convert MP4, MOV, or WEBM video files to .txv in the browser. Live preview while converting. Configurable columns, FPS, duration, contrast, and colour mode.

↓ download
TXV CONVERT python
txv-convert.py

Command-line converter using OpenCV. Fast frame seeking, progress bar, full metadata support. Requires opencv-python and numpy. Supports any video format OpenCV handles.

↓ download

Get Started

Up in three steps.

01

Download the tools

Download all four files above and put them in the same folder.

ascii-cam.html
txv-player.html
txv-converter.html
txv-convert.py
02

Start a local server

Chrome requires a local server for webcam access. Python's built-in server works perfectly.

cd ~/Downloads
python3 -m http.server 8080
03

Open in Chrome

Navigate to localhost and open any tool. Allow camera access when prompted.

http://localhost:8080/
ascii-cam.html

Python Converter

# install
pip install opencv-python numpy

# basic conversion
python3 txv-convert.py myvideo.mp4

# high resolution, 24fps, edge mode
python3 txv-convert.py clip.mp4 --cols 120 --fps 24 --mode edge

# with metadata
python3 txv-convert.py clip.mp4 --author yourname --note "first txv"
